# La Via della Solitudine — Dokkōdō
### 独行道

Diario operativo. Sistemi minimi costruiti sul campo.

Non è un portfolio tradizionale. Non è un blog motivazionale.
È una raccolta di artefatti nati dalla gestione quotidiana di processi complessi, in ambienti imperfetti e spesso non strutturati.

Il metodo è sempre lo stesso: osservare il caos, trovare il punto minimo di intervento, rendere tracciabile ciò che prima non lo era.

## Il riferimento

Il Dokkōdō è una raccolta di 21 precetti scritti da Miyamoto Musashi una settimana prima di morire.
Non è filosofia motivazionale: sono regole asciutte per agire con disciplina quando il contesto non collabora.

> *"Non fare nulla che non sia di utilità."*
> — Miyamoto Musashi, 1645
